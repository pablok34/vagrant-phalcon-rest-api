# Phalcon RESTful API

## Installation

Clone repository:

```BASH
$ git clone https://github.com/pablok34/vagrant-phalcon-rest-api.git
$ cd vagrant-phalcon-rest-api
```

Vagrant:

```BASH
$ vagrant up
```

Configure in your host file (sudo vi /etc/hosts in Mac)

```BASH
    10.4.4.52 vm01.devel
    10.4.4.52 phalcon.devel
```
